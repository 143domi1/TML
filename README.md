# TML

TML (Teardown mod loader) is a **simple** mod loader for the game Teardown.  
This program is NOT officially endorsed or supported by the Teardown developers.

## Features
TML can currently:
1. Install mods in seconds

## About
But for the lack of features its very fast, easy to use.  
And you can use this instead of manually copying the required folder into the correct path.  

Official methods do exist! But it's best to also have community options as well.
